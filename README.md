# retail-profitability-analysis
End-to-end retail sales and profitability analysis using Python, Pandas, and Jupyter Notebook.

This project will simulate a project for a jr. financial analyst, we want to evaluate the last few years of sales data and determine the answers to key financial questions:

- Which categories generate the most profit?
- Which regions underperform?
- What seasonal trends exist?
- What recommendations should leadership consider?
